# Ozon Parser

Этот проект представляет собой многопоточное приложение для извлечения информации о товарах с сайта Ozon. Используются Selenium для автоматизации браузера и Python для управления потоками.

## Установка

1. Установите Python (версия 3.7 или выше).
2. Установите необходимые пакеты:

   ```bash
   pip install selenium webdriver-manager
